# Amazon Product Review Sentiment Analysis

A Machine Learning project that classifies Amazon product reviews as **Positive** or **Negative** using Natural Language Processing (NLP).

## Technologies Used
- Python
- Pandas
- NLTK
- Scikit-Learn
- TF-IDF Vectorizer
- Logistic Regression

## Features
- Text Preprocessing
- Stopword Removal
- TF-IDF Vectorization
- Sentiment Classification
- Model Evaluation

## Results
- Accuracy: 81.6%
- Binary Sentiment Prediction

## Dataset
Amazon Product Reviews Dataset (~25,000 reviews)

## Future Improvements
- BERT-based Sentiment Analysis
- Streamlit Deployment
